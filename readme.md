# TensorFlow 2.0 Tutorials
A collection of TensorFlow programs, following the official [TensorFlow 2.0 Tutorials](https://www.tensorflow.org/alpha).

## Requirements
The scripts require the following libraries to run:
* Future
* Keras
* NumPy
* Pandas
* MatPlotLib
* SeaBorn
* SKLearn
* Time
* OS
* TensorFlow 2.0
* TensorFlow Datasets
